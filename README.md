# BeatModsInstaller

The 'BeatModsInstaller' is a simple tool to downlaod mods from the site 'https://beatmods.com'.

## Features
- Install multiple mods at once with dependencies

## Upcoming features
- Search mods
- Download songs
- Search songs
- Categorize mods
- Download queue
- Installed mods
- Active/Deactive mods
- List dependencies under the mods (expendable).
